# Case Check

It's easy to check what case that string used by
**vanilla methods:** `toUpperCase` and `toLowerCase`.

## Source Code

```js
function isUpperCase(str) {
  return str === str.toUpperCase();
}

function isLowerCase(str) {
  return str === str.toLowerCase();
}
```

<Callout type="warning">
  You should assert param `str` is a `string` before invoke String methods.
</Callout>
